🐠 OSHI FISH Aquarium Website

A floating aquarium-style social feed built with HTML, JavaScript, and CSS.

## Features
- Upload images/videos
- Add captions
- Aquarium animated background
- Messenger redirect button
- Owner-only posting system
- Delete posts feature

## Setup
1. Upload files to GitHub repo
2. Enable GitHub Pages
3. Open index.html
